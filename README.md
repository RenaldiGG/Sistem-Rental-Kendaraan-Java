# Sistem Rental Kendaraan Berbasis Java

## Deskripsi
Program ini merupakan proyek UAS Pemrograman Berorientasi Objek (PBO) yang dibuat menggunakan bahasa pemrograman Java.

## Fitur
- Menampilkan daftar kendaraan
- Menambah kendaraan
- Mengedit kendaraan
- Menghapus kendaraan
- Login admin
- Penyewaan kendaraan
- Proses pembayaran
- Exception handling

## Konsep OOP yang digunakan
- Class & Object
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interface
- Exception Handling
- Collection Framework
